PSYBER PORTAL — WEEK-UNLOCK EMAILS
==================================

Each file here is the email a client receives when you move them forward to
that week in the admin dashboard (once the week is published). Edit the text,
tell Claude to deploy, and your changes go live.

FORMAT (keep it simple)
-----------------------
- The FIRST line must start with "Subject:" followed by the subject line.
- Leave a blank line, then write the body.
- Separate paragraphs with a blank line. Each block becomes one paragraph.
  (You don't need to worry about line length — a single blank line is what
  starts a new paragraph.)
- Don't add the greeting, the "Open your portal" button, the "just reply"
  line, or the sign-off — those are added automatically around your text.
  Just write the middle: encourage them, remind them what the last session
  covered, say what this week is about, and give your advice for the week.

ADDING A NEW WEEK
-----------------
- Create week-03.txt (then week-04.txt, and so on) in the same format.
- A week with no file here sends NO email — never a generic one.

NOTES
-----
- Emails are signed "Asher"; replies go to asher@psyberdigital.com.
- Use US spelling (the audience is US therapists): "program", not "programme".
- File names must be exactly week-01.txt, week-02.txt, etc. (two digits).
